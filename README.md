## 아이디어 소개 및 팀 구성
```
진공도 감지 센서 활용 및 분석을 통한 진공장비의 예지보전 키트 VACMA KIT
(Vaccum Chamber Condition-based Maintenance)
```

- 한국산업기술대학교 19' 전자공학과 김남균
